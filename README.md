# TaskMaster 📝

Um gestor de tarefas simples, rápido e moderno construído inteiramente com **HTML**, **CSS** e **Vanilla JavaScript**. 

Este projeto foi desenvolvido para consolidar os fundamentos de desenvolvimento web (manipulação do DOM, eventos e persistência de dados no navegador).

## 🚀 Funcionalidades

- **Adicionar Tarefas:** Escreve o que precisas de fazer e prime `Enter` ou clica no botão.
- **Marcar como Concluída:** Clica no texto da tarefa para a riscar da lista.
- **Eliminar Tarefas:** Remove tarefas que já não precisas com um simples clique.
- **Persistência de Dados (LocalStorage):** As tuas tarefas ficam salvas no teu próprio navegador. Podes fechar a página ou atualizar que não perdes nada!

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do projeto.
* **CSS3:** Estilização moderna utilizando Flexbox, hover effects e transições suaves.
* **JavaScript (ES6+):** Lógica da aplicação, manipulação do Document Object Model (DOM) e integração com a Web Storage API (`localStorage`).
